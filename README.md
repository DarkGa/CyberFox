## Install CyberFox
* 1. copy file
```
git clone https://github.com/DarkGa/CyberFox
```
* 2. open directory CyberFox
```
cd CyberFox
```
* 3. install setting
```
chmod +x *
```
```
./install.sh
```
* 4. start CyberFox
```
cf
```
